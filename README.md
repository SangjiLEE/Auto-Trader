# Auto Trader

> **혼자서 8일 만에 만든 멀티 전략 자동매매 시스템.**
> Python + Claude Code (AI 페어 프로그래밍) + KIS Open API.

한국주식 + 미국주식을 평일 자동으로 매매. 시장 체제 (BULL/RANGE/BEAR) 를 매일 분석해서 다른 룰을 적용. 공포탐욕지수 극단치에 분할매매. 모든 거래는 Telegram 으로 실시간 알림.

**현재 모의투자 계좌에서 자동 운영 중** (실거래 전환 전 3-6개월 검증 단계).

## 왜 만들었나

### 처음 목표
> "AI 사용 구독료 (월 30만원) 를 자동매매 수익으로 충당하고 싶다."

### 진짜 목표 (검증 후 수정)
> "**손해 가능성을 체계적으로 줄이면서 점진적으로 수익률 개선되는 개인용 수익 파이프라인.**"

8일 동안 데이터로 발견한 진실:
1. **100만원으로 월 30만원은 수학적으로 불가능** (필요 월수익률 30%)
2. **모든 전략이 단순 보유 (Buy & Hold) 를 절대값으로 못 이김** — 장기 강추세 시장에서
3. **하지만 MDD (최대 낙폭) 는 절반 이하**로 줄일 수 있음
4. → "잘 안 잃기" 가 "더 잘 벌기" 보다 현실적

### 결론
- **자본 규모 작을 땐 학습 + 인프라 구축 비용**으로 받아들임
- **6~12개월 모의 검증** 후 자본 증액 검토
- 시스템을 잘 만들어두면 **3년 후 자본 증가 시 즉시 적용 가능**

## 무엇을 썼나 (Tech Stack)

| 영역 | 도구 |
|-----|-----|
| **언어** | Python 3.12 |
| **증권사 API** | 한국투자증권 KIS Open API (REST + OAuth2 + hashkey) |
| **시세 데이터** | FinanceDataReader (FDR), Yahoo Finance |
| **DB** | SQLite |
| **분석** | pandas, numpy |
| **알림** | Telegram Bot API |
| **공포탐욕지수** | Alternative.me F&G API |
| **자동화 스케줄** | macOS launchd (5개 plist) |
| **개발 환경** | Claude Code (AI 페어 프로그래밍), VS Code, zsh |
| **시스템** | macOS (Apple Silicon) |

**의도적으로 안 쓴 것:**
- 외부 백테스트 라이브러리 (`backtrader`, `vectorbt`) — 룰을 직접 짜는 게 학습에 더 좋음
- 실시간 분봉 (intraday) — 일봉만으로도 충분한 검증 가능
- 도커 / 클라우드 — 로컬 launchd 가 더 단순

자세한 진화 과정: [docs/JOURNEY.md](docs/JOURNEY.md)

### 개발 방식: Claude Code + AI 페어 프로그래밍

**Claude (Anthropic) 와 Pair Programming 방식.**

- **사용자**: 의도, 도메인 지식, 의사결정, 검증
- **Claude**: 코드 작성, 디버깅, 백테스트 설계, 테스트
- **검증 사이클**: 모든 코드는 사용자가 직접 실행 + 결과 확인 + 다음 결정

이 방식으로 8일에 ~7,000줄 코드 + 30+ 모듈 구축. **AI 가 코드를 쓰고, 사람이 의사결정 + 검증.**

### 하네스 엔지니어링 (Harness Engineering)

이 시스템의 핵심은 **반복 가능한 검증 인프라** (= 하네스).

자세한 내용: [docs/HARNESS.md](docs/HARNESS.md)

요약:
1. **백테스트 하네스 5개** — 같은 패턴으로 새 전략 빠르게 검증
2. **견고성 검증 하네스** — 파라미터 민감도 + OOS + 연도별 자동
3. **전략 플러그인 아키텍처** — strategy 모듈 교체만으로 다른 룰 시도
4. **체제 분류 하네스** — BULL/RANGE/BEAR 단일 진실 출처
5. **드라이런 ↔ 실행 토글** — `--execute` 한 플래그로 안전 / 운영 분리
6. **상태 재구성 하네스** — DB trades 에서 포지션·체제·부분익절 상태 모두 복원
7. **5개 launchd 작업 분산** — 한 작업 실패해도 다른 작업 영향 X

이 하네스 덕분에 8일간 **5번의 큰 전략 재설계** (slow → fast → v3 → v4 → relaxed v4) 가능.

## 빠른 시작

```bash
git clone https://github.com/SangjiLEE/Auto-Trader.git
cd Auto-Trader
python3.12 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cp .env.example .env  # KIS API 키, Telegram 토큰 입력
nano .env

python -m src.hello_world  # KIS 인증 테스트
python -m src.daily_swing_v3_kr  # KR 스윙 드라이런
python -m src.swing_backtest_v4   # 백테스트
```
