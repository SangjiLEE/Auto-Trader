## 1. Dual Momentum (DM) — 70% 슬롯, 월간

Gary Antonacci (2014, *Dual Momentum Investing*) 의 **상대 + 절대 모멘텀** 결합.

### 유니버스

한국 상장 ETF만 사용. 환전·해외세 회피 + KIS 국내주식 API 단일화.

| 종목코드 | 이름 | 노출 |
|---------|------|-----|
| `069500` | KODEX 200 | 한국 주식 |
| `133690` | TIGER 미국나스닥100 | 미국 기술주 |
| `360750` | TIGER 미국S&P500 | 미국 광범위 |
| `148070` | KOSEF 국고채10년 | 위기 방어 |

### 룰

```
매월 말:
  1. 각 자산의 12개월 누적 수익률 계산
  2. 가장 높은 수익률 자산 선택
  3. 그 수익률이 0% 초과면 → 다음 달 그 자산 100% 보유
  4. 모두 0% 이하면 → 현금 (CASH)

매월 첫 영업일 09:05:
  5. 자동으로 타겟 자산으로 리밸런싱 (DM_ALLOCATION = 70%)
```

### 검증 결과 (2020-08 ~ 2026-04, 5.7년)

- CAGR **+23.78%** / Sharpe **1.10** / MDD -22.6%
- 069500 BH 대비: CAGR -0.7%p, MDD **12%p 개선**
- 모든 BH 대비 Sharpe 우수

### 견고성 검증 통과 항목

- **파라미터 민감도**: 룩백 3~24개월 모두 CAGR 18~24% (안정)
- **IS/OOS 분할**: 2020 이전 학습 vs 이후 검증 — 일관성 확인
- **연도별**: 음수 연도 4/11 (정상 범위), 최악 -13.47%

## 2. Enhanced Swing v3 — 체제별 어댑티브 (KR/US 각 15%)

같은 종목이라도 **시장 체제 (Regime) 에 따라 룰을 변경**.

### 시장 체제 자동 분류 (`market_regime.py`)

```python
def detect_regime(row) -> str:
    if close > ma200 * 1.05 and ma60 > ma200 and adx > 20:
        return BULL
    if close < ma200 * 0.95 and ma60 < ma200 and adx > 20:
        return BEAR
    return RANGE
```

ADX > 20 조건으로 노이즈 제거.

### 진입 조건 (4 AND, 모든 체제 공통)

1. 종가 > 20MA (단기 상승)
2. 전환선 > 기준선 (일목 강세)
3. RSI(14) ∈ [30, 75] (과매수 회피)
4. 거래량 > 20일 평균 × 1.2

추가 (BEAR): 진입 자체 차단.

### 체제별 차별 룰

| 항목 | BULL | RANGE | BEAR |
|-----|-----|-----|-----|
| 초기 진입 사이즈 | 70% | 50% | **0% (차단)** |
| 손절 | -5% | -3% | - |
| 1차 부분익절 | (없음) | +3% (30% 매도) | - |
| 2차 부분익절 | +20% (30% 매도) | +7% (30% 매도) | - |
| 트레일링 진입 | +30% | +12% | - |
| 트레일링 폭 | -5% | -3% | - |
| 20MA 이탈 청산 | ✓ | ✗ | - |
| 시간 손절 | 30일 | 15일 | - |
| DCA 트리거 | -2% | -1.5% | - |
| 재진입 쿨다운 | 3일 | 1일 | - |

### 핵심 아이디어

- **BULL**: 큰 흐름 잡기. 긴 보유, 큰 익절. 부분익절 늦게.
- **RANGE**: 빠른 회전. 작은 익절 자주 (사용자 직관 — 오르면 팔고 떨어지면 사기).
- **BEAR**: 자본 보호 우선. 진입 안 함.

### 검증 결과

| 종목 | 기간 | v3 누적 | BH | MDD | Sharpe |
|-----|-----|--------|-----|-----|-------|
| NVDA | 5년 | +64.2% | +1508% | -17.0% | 0.67 |
| TSLA | 5년 | +55.0% | +85% | -17.7% | 0.68 |
| 069500 | 10년 | +19.1% | +378% | -17.6% | 0.29 |
| 005930 | 10년 | +18.9% | +775% | -37.4% | 0.22 |

### 발견된 한계

- **장기 강추세 종목 (NVDA 5년 +1508%)** 은 BH 압도적
- v3 부분익절이 큰 흐름 끊음
- **MDD 절반** 으로 변동성 견딤 (Sharpe 비슷)
- 단기 (1-5년) + 적당한 변동성에 가장 잘 맞음

## 3. F&G Integration (v4 relaxed)

[Alternative.me F&G API](https://alternative.me/crypto/fear-and-greed-index/) 활용.

### 일반 시장 (F&G 7~91): 사이즈 배율

```python
def position_size_modifier(fg_value):
    if fg_value < 20:  return 1.30   # Extreme Fear, 저점 매수 강화
    if fg_value < 30:  return 1.15   # Fear
    if fg_value > 80:  return 0.00   # Extreme Greed, 진입 차단
    if fg_value > 70:  return 0.50   # Greed
    return 1.00                       # Neutral
```

진입 시 v3 의 `initial_buy_ratio × fg_modifier` 로 최종 사이즈 결정.

### F&G 극단 분할매매

```python
F&G ≤ 7 (Extreme Fear):
  → 매일 슬롯 cash 의 25% 분할매수
  → 첫 universe 종목으로 (069500 / AAPL)
  → 4일에 걸쳐 매수 완료

F&G ≥ 92 (Extreme Greed):
  → 매일 보유의 10% 분할매도
  → 단, BULL 모드 포지션은 제외 (v3 BULL 익절 우선)
  → 10일에 걸쳐 매도
```

### 룰 완화 사유

초기 v4 strict (≥88, 25%) 백테스트:
- TSLA: +38% (v3 +55% 보다 -17%p)
- 069500: +10% (v3 +19% 보다 -9%p)

→ 강세장에서 너무 자주 매도. 룰 완화 후:

v4 relaxed (≥92, 10%, BULL skip):
- TSLA: +56% (v3 동급)
- **069500: +28.85%** (v3 +19% 대비 +9.7%p ⭐)

## 데이터 의존성

### F&G 한계

Alternative.me F&G 는 **암호화폐 sentiment** 기반. 글로벌 위험선호도 프록시로 사용.

- 한국 시장 전용 F&G 미사용
- CNN 의 미국 주식 F&G 는 공개 API 없음
- VIX 기반 자체 sentiment 추후 구현 가능

### 시세 데이터

- **FinanceDataReader** (FDR) — 한국 + 미국 일봉
- **KIS API** — 실시간 시세 (장중 매매 시)
- 분봉 / 인트라데이 미사용 (일봉 기준만)
