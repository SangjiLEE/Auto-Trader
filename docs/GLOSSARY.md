# 용어집 (Glossary)

영어로 된 코드 / Strategy 이름 → 한글 표현 정리.

> **운영 원칙**: DB / 코드의 strategy 컬럼은 영어 (ASCII) 유지.
> Telegram 알림 + 보고서 + 사용자 대화에서만 한글 표시.

## 🎯 운영 중인 Strategy (전략)

|-----------|---------|----------|

## 🔑 룰 / 메트릭 용어

| 영문 | 한글 | 의미 |
|-----|------|------|
| Trailing Stop | 추격 손절 / 따라가는 청산 | 최고점에서 -N% 떨어지면 청산 |
| Profit Cap | 익절 상한 / 수익 잠금 | +10% 도달 시 무조건 청산 |
| Cooldown | 재진입 대기 | 매도 후 N일 동안 같은 종목 금지 |
| DCA (Dollar Cost Averaging) | 분할 매수 / 평단 낮추기 | 손실 시 추가 매수로 평단 ↓ |
| PEAD | 실적 발표 후 표류 | 발표 후 1~3일 추세 지속 학술 효과 |
| Catalyst Window | 호재 구간 | 실적 발표 ±N일 매수 가능 시기 |
| Profit Trigger | 익절 트리거 | +N% 도달 시 trailing 활성화 |
| Stop Loss | 손절 | -N% 도달 시 자동 청산 |
| Time Exit | 시간 만료 청산 | N일 보유 후 무조건 청산 |

## 💹 성과 메트릭

| 영문 | 한글 | 의미 / 해석 |
|-----|------|-----------|
| **CAGR** | 연복리 수익률 | 매년 평균 % 복리로 자본 늘어남 |
| **Sharpe** | 샤프 (위험조정 수익률) | 변동성 1단위 당 수익. 1.0+ 양호, 2.0+ 우수 |
| **Calmar** | 칼마 (낙폭조정) | CAGR / |MDD|. 0.5+ 양호, 1.0+ 우수 |
| **MDD** | 최대 낙폭 | 고점 대비 가장 큰 손실 (음수) |
| **Underwater** | 손실 회복 대기일 / 수면 아래 | 고점 → 회복까지 일수 |
| **Win Rate** | 승률 | 양수 거래 / 전체 거래 |
| **EV** (Expected Value) | 기대값 | 거래 1회당 평균 P&L |
| **Drawdown** | 낙폭 / 일시적 손실 | 고점 대비 현재 손실 (%) |
| **Ulcer Index** | 궤양 지수 | 낙폭 심각성 누적 (낮을수록 좋음) |

## 🔧 운영 / 인프라 용어

| 영문 | 한글 | 의미 |
|-----|------|------|
| **Universe** | 종목군 / 매매 대상 | 거래 가능한 종목 목록 |
| **Allocation** | 자본 비중 / 슬리브 비중 | 전체 자본의 N% 배정 |
| **Sleeve / Slot** | 슬리브 / 운용군 | 한 strategy 가 쓰는 자본 영역 |
| **Fill Check** | 체결 확인 / 잔고 대사 | 주문 후 잔고 차분으로 검증 (B2 fix) |
| **Reconciliation** | 잔고 대사 / 매칭 | DB ↔ 증권사 잔고 일치 확인 |
| **Backtest** | 백테스트 / 과거 검증 | 과거 데이터로 룰 시뮬레이션 |
| **Walk-Forward** | 전진 검증 | 시간 순으로 학습→검증 반복 |
| **Look-Ahead Bias** | 미래 정보 누설 | 과거 시점에 미래 데이터 사용 (오류) |
| **Slippage** | 미끄러짐 / 체결가 손실 | 주문가 vs 실제 체결가 차이 |
| **F&G** | 공포탐욕지수 | 시장 심리 0-100 지수 |
| **DM** | 이중 모멘텀 (Antonacci) | 상대 + 절대 모멘텀 결합 |
| **TAA** | 전술적 자산배분 | Tactical Asset Allocation |
| **plist** | 자동 작업 설정 | macOS launchd 스케줄 파일 |
