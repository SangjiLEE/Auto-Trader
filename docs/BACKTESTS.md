## 3. v4 (v3 + F&G 극단 분할매매)

### v3 vs v4 strict vs v4 relaxed 비교

**룰:**
- strict: F&G ≤ 7 매수 25%, F&G ≥ 88 매도 25%, BULL 무관
- relaxed: F&G ≤ 7 매수 25%, F&G ≥ 92 매도 10%, **BULL skip**

| 종목 | v3 | v4 strict | v4 relaxed |
|-----|-----|-----------|-----------|
| AAPL (2년) | +4.1% | +0.6% | +0.6% |
| NVDA (5년) | +64.2% | +57.4% | +57.4% |
| TSLA (5년) | +55.0% | +38.4% | **+56.3%** |
| **069500** (10년) | +19.1% | +10.1% | **+28.85%** ⭐ |
| 005930 (10년) | +18.9% | +8.1% | +18.5% |

**핵심 발견:**
- v4 strict 가 BULL 강세장에서 너무 자주 매도 → 손실
- v4 relaxed 가 v3 와 동급 또는 더 우수 (069500 +9.7%p)
- 사용자 직관 ("공포에 매수, 과열에 매도") 살리는 균형점

## 핵심 학습

1. **모든 전략이 BH 절대값 못 이김** (장기 강추세 시장에서)
2. **MDD 절반 이하**는 가능 (Sharpe 비슷, "잘 못 잃기")
3. **체제별 어댑티브 (v3) > 단일 룰**
4. **F&G 통합 (v4 relaxed) > v3 단독** (일부 종목)
5. **단타 / 평균회귀는 우리 setup 에서 효과 없음** (데이터 증거)
6. **유니버스 선별이 자체 전략보다 더 중요할 수도** (강추세 종목 제외)
